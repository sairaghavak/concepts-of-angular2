Dependency Injection

  • Angular has inbuilt Dependency Injection framework and it injects all the services that are required for us upon registering them.
  • Services can be registered in 2 ways.
    1. In @NgModule decorator's providers array. Here a single instance of the service is created at application start-up time and can be used across the module.
    2. In @Component decorator's providers array. Here the service instance is created every time component is constructed, and its usage is restricted to the correspondingcomponent.
