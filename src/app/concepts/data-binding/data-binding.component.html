DataBinding

  • A concept of binding data between templates and components.

  • Data binding can be done in 3 ways.

    1. Component to Template

      • Here data is sent from component in 2 ways again.

      • Interpolation

        • Examples:
          • {{ interpolationSnippet1 }}
            Output:
             {{ interpolationSnippet1Output }}
          • {{ interpolationSnippet2 }}
            Output:
             {{ interpolationSnippet2Output }}
          • In short, it functions like System.out.println() in Java.
            Example: {{ interpolationSnippet3 }}
            Output:
             {{ interpolationSnippet3Output }}

      • Property Binding

        • Examples:
          • {{ pbSnippet1 }}
            Output:
          • {{ pbSnippet2 }}
            Output:

          {{ pbcssCodeNote }}

          • {{ pbcssSnippet1 }}
            Output:
            Inline display..
          • {{ pbcssSnippet2 }}
            Output:
            z to a

          {{ pbhtmlAttNote }}

          • {{ pbhtmlAttSnippet1 }}
            Output:
            a to z
          • {{ pbhtmlAttSnippet2 }}
            Output:

    2. Template to Component

      • Here data is sent from template to component.
      • This is also known as Event Binding.
      • Example:
        • {{ tcSnippet1 }}
          Output:

    3. Component to Template and Vice versa

      • This is commonly know as 2 way binding.
      • Example:
        • {{ ctSnippet1 }}
          {{ ctSnippet2 }}
          Output:

          {{ defaultTextMessage }}
