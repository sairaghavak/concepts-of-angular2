Modules

  • A module in Angular is just another Typescript class.
  • Angular recognizes the class as angular mdodule when it is decorated with a decorator called @NgModule. These decorators are similar to annotations in Java. Example: @Override.
  • Every angular application will have at least one angular module. The default angular module is a root angular module and is conventionally named as AppModule in Angular applications.
